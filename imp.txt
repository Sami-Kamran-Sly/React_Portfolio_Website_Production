- **blurhash**: A library that generates a compact representation of an image to display as a blurred placeholder while the full image loads.
- **framer-motion**: A library for adding smooth, declarative animations and interactions to React components.
- **react-blurhash**: A React component for rendering images as blurhash placeholders before they fully load.
- **react-icons**: A React library that allows you to easily include a wide variety of icons from popular icon sets in your app.
- **react-router-dom**: A React library for handling routing and navigation between different pages or components in web applications.
- **react-simple-typewriter**: A lightweight React component that creates a typewriter effect for text animations.



Packages To Install : npm install blurhash framer-motion react-blurhash react-icons react-router-dom react-simple-typewriter
